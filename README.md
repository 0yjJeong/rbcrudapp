# rbcrudapp

> react-based CRUD app

## Getting started

Open a terminal window and start the web app using following commands from the project root:

### Running Locally

```bash
$ yarn
$ yarn build
$ yarn start
```

### Usage

```tsx
import React from 'react';
import { Admin } from '@rbcrudapp/core';
const App = () => {
  return <Admin id='id' />;
};
```
